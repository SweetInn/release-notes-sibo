# Version 47 Additionals

**Released:** Jun 20, 2024

### What's New?

#### <u>OCI (Online Check-In)</u>

**Check-in & Check-out step:**
- Re-ordered steps — this step is now placed after the Passports step.
- Renamed to "Check-in & Check-out".
- Departure time is now collected as well.
- Guests can now request early check-in.
- Guests can now request late check-out.
  *(Paid Early and Late services will be added in the next version)*
- New hours picker for ETA and ETD.
- Removed the last-minute ETA update limit.
- Removed the Flight number option from this step.

**Thank you step:**
- An additional message is now shown when Online Check-In is completed more than 3 days prior to check-in:
  *"We will send you the check-in instructions by email 3 days prior to your arrival."*

#### <u>SIBO Unified-Inbox</u>

- New auto-rules capability — conversations can now be automatically assigned to a specific team or user based on the same rules system.
