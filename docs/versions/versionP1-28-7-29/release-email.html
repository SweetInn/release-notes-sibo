Release Notes - April 2026 (Version 28.7 + 29)

Dear Sweett Team,

In this release:

  • SIBO Mobile App is now in your pocket, with core operational flows and real-time alerts for field teams.
  • Service pay-links can now be completed in person via POS terminal or cash.
  • Recurring tasks and cleanings enable teams to automate repetitive operations with recurring schedules in SIBO.
  • Alteration link extensions now cover direct pre-check-in and Booking.com in-stay flows.
  • Pricing updates now include a pre-apply summary showing exactly what will change before submission.
SIBO Mobile App
SIBO mobile launches with day-to-day operational capabilities and instant last-minute booking notifications for faster field response.
Service Pay-Links with Offline Payment Reporting
Service pay-links can now be settled in person using a POS terminal card payment or cash, improving front-desk flexibility.
Recurring Tasks & Cleanings
Teams can define recurring schedules and let SIBO automatically generate repetitive tasks and cleanings.
Alteration Link Extensions
Direct guests can extend before check-in, and Booking.com guests can extend in-stay through an adapted extension flow.
Pricing Change Impact Summary
A new pricing impact summary is shown before apply, helping users verify what is about to change and avoid unintended updates.

Website & App

Online Check-In

  • Two-Sided Document Scan in OCI
    During online check-in, guests may be asked to scan both sides of required documents in markets where double-sided validation is required.

Alteration & Booking Changes

  • Pre-Check-In Night Extension for Direct Guests
    Direct guests can extend their stay before check-in, complementing the existing checkout-side extension journey.
  • Booking.com Alteration Coverage Expanded
    Extension links now handle both pre-arrival and in-stay Booking.com extension scenarios.

SIBO

Digital Access

  • Digital Keys for Maintenance Tasks
    SIBO now supports end-to-end digital-key handling for maintenance tasks, including key lifecycle orchestration and access visibility for assigned users.

Direct Sales & Services

  • Service Pay-Links with Offline Payment Reporting
    When payment is taken outside the pay-link flow, teams can record the transaction in SIBO after collecting funds through channels like card terminals, bank transfer, or cash.

Task & Cleaning Management

  • Recurring Tasks and Cleanings
    Teams can define recurring schedules and let SIBO automatically generate repeating tasks and cleanings.
  • Planner Configuration Alerts for Cleaning Managers
    The daily planner surfaces missing setup issues directly in context, such as absent default assignees, companies, durations, or costs.
  • Cleaner Activity Tracking
    SIBO records whether cleaning tasks were started or finished manually, triggered by the lock system, or timed out, giving operations teams accurate visibility into field activity.

SIBO App

  • SIBO Mobile App
    SIBO mobile launches with core day-to-day operational capabilities for faster field execution.
  • Push Alerts for Last-Minute Bookings
    Operations teams receive app push notifications for last-minute bookings so they can react quickly.

Channel Management

  • Airbnb Maximum Advance Booking Window
    Data teams can configure how far ahead guests may book Airbnb listings directly from SIBO channel settings.

Guest Experience & Scheduling

  • LCO Auto-Approval Aligned with Maintenance
    Late check-out daily auto-approval now accounts for relevant maintenance tasks to reduce downstream scheduling conflicts.

Booking Management

  • Pricing Change Impact Summary
    Before applying pricing updates, users now see a summary of planned changes and affected properties.

Property & Unit Management

  • Properties and Units Renaming
    Hosts can rename both properties and units so labels shown internally and in guest communication remain accurate after branding or data updates.
  • Default Amenities Set Support
    Accounts and cities can define default amenities that are automatically applied when creating new properties or units.

Enhancements & Bug Fixes

  • Enhancement Rentals United Updates Without CredentialsSIBO
    Availability and price updates to Rentals United now handle accounts that currently have no stored credentials.
  • Enhancement SIBO App Update EnforcementSIBO
    SIBO App can now show mandatory blocking updates or recommended non-blocking update prompts based on release policy.
  • Enhancement Configurable Key Access BufferSIBO
    Operations teams can now set digital-key access buffers per account and city, replacing the previous single global timing rule.
  • Enhancement Truvi Flagged Coverage DisplaySIBO
    When Truvi flags a booking, hosts now see in the Insurance card that Damage Protection is capped up to 1,000 EUR.
  • Enhancement Conditional Alteration Pay-LinkSIBO
    The alteration finish flow now shows pay-link actions only when unpaid dues exist, while preserving direct sidebar access when payment is required.
  • Enhancement OCI Time Picker ClarityWebsite/App
    Check-in and check-out time pickers now animate briefly when opened to make scroll selection clearer to guests.
  • Enhancement No Lock Data Loaded for Past BookingsWebsite/App
    The website and app no longer load door access data when viewing past bookings, reducing unnecessary requests and related errors.
  • Bug Fix Agoda Non-Refundable DetectionSIBO
    Non-refundable Agoda reservations are now correctly identified so proper cancellation policy behavior is preserved.
  • Bug Fix Minimum-Stay Publish TriggerSIBO
    Minimum-stay-only pricing changes now trigger channel publishing and sync to connected providers.
  • Bug Fix Airbnb Booking IngestionSIBO
    Airbnb bookings now process even when the property is not yet published on the website.
  • Bug Fix ECI/LCO Invalid Date DisplaySIBO
    Auto-confirmed ECI/LCO entries now show the correct confirmed time instead of an invalid date value.
  • Bug Fix Unintended Alteration Pay-Link SendSIBO
    Alteration pay-links are no longer auto-sent to guests unless explicitly intended by staff flow.
  • Bug Fix Canceled Booking Search StatusSIBO
    Booking search results now show the correct status after cancellations that include prior guest ID uploads.
  • Bug Fix Report Form Required Fields GateSIBO
    The Report action button now stays disabled until all mandatory questions are completed.
  • Bug Fix Dashboard Accommodation CoverageSIBO
    SIBO Dashboard accommodation filters now include building and unit types in unsold-apartments counts.
  • Bug Fix Salto Code with Special CharactersSIBO
    Door-code creation now handles guest names that include invisible special characters from channel sources.
  • Bug Fix Pre-Payment Change ClassificationSIBO
    Booking adjustments made before payment completion are no longer incorrectly logged as alterations.
  • Bug Fix Bookings Total After Full RefundSIBO
    Paid/Total values on the Bookings page now show zero correctly after full refunds with policy overrides.
  • Bug Fix Delivery Task Cancellation RefundSIBO
    Canceling paid delivery tasks no longer triggers incorrect refunds and residual booking balances.
  • Bug Fix Admin Code Action VisibilitySIBO
    The Regenerate Admin Code action is no longer exposed on booking lock view where it should not appear.
  • Bug Fix Cancellation Max Refund PrecisionSIBO
    Refund-limit calculations now avoid floating precision artifacts and allow exact remaining-balance refunds.
  • Bug Fix Post-Partial Cancellation Refund LogicSIBO
    Cancellation refunds now correctly account for earlier partial refunds and return only the true remaining balance.
  • Bug Fix Invoice View Loading FailureSIBO
    Invoice view now loads properly for bookings that previously returned display errors.
  • Bug Fix Breakfast Dashboard EligibilitySIBO
    Only hotels with active breakfast offerings now appear in the Breakfast Management dashboard.
  • Bug Fix Scheduled Installment Auto-CancelSIBO
    Scheduled installments are now canceled automatically once the full booking balance is paid.
  • Bug Fix Tasks Settings Column ScopeSIBO
    Auto-generated inventory checklist sub-types are no longer shown as configurable columns in task settings.
  • Bug Fix Maintenance Smart Code RevocationSIBO
    Smart lock codes for maintenance visits are now revoked automatically when the task is marked done.
  • Bug Fix Checkout Instructions Time PlaceholderWebsite/App
    Checkout instruction copy now shows real lobby hours instead of a placeholder token.
  • Bug Fix OCI Auto-Translation CrashWebsite/App
    Online Check-In no longer crashes when guests use browser auto-translation features.
  • Bug Fix Duplicate OCI Tab Paid ECI OverrideWebsite/App
    A second OCI tab submission can no longer overwrite a previously confirmed paid early check-in slot.
  • Bug Fix OCI Damage Protection Step ProgressionWebsite/App
    After Damage Protection payment, guests now proceed correctly to the next OCI step without manual refresh.
