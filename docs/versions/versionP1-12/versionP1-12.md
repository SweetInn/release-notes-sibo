# Version P1.12

**Released:** Nov 19, 2024

![Version P1.12 - New Guest Services]({{ "/assets/images/versionP1.12/2026-03-09_12-03-54.png" | relative_url }})

### What's New?

#### <u>Website & App</u>

**Guest Services:**
- Extend Your Stay — new self-service feature for guests to extend their stay directly.
- Transportation Services — guests can now order check-in or check-out driver services.
- Cleaning Service — self-service option added for guests to schedule in-stay cleanings.
- Service Tiles — dedicated tiles for Transportation, Cleaning, and Extension services, promoting relevant services when available.

**Hotels Support:**
- Hotel Directory Page — displays guest information including dining schedules, WiFi details, and general information sourced from SIBO.

#### <u>SIBO</u>

**Operations Platform:**
- Task Organizer View — new view to streamline task assignments and scheduling for city employees.
- Mobile Task Visuals — clear visual distinctions for tasks scheduled on different dates, enabling cleaners and users to navigate daily tasks easily.
- Task Notification Control — email notifications on task assignments/re-assignments will now only trigger if the task occurs the same day.
- Postponed Tasks — postponed reason will be removed when a task is rescheduled.
- Duplicate Report Prevention — existing open reports are now displayed to reduce duplicate report submissions.

**Digital Keys:**
- Salto Lock Admin Code Regeneration — added functionality to regenerate admin codes for digital Salto locks in hotels.

**Bookings Management:**
- Direct Links for Additional Services — direct links to new guest services are now accessible in booking details for seamless ordering on the website and app.

**Content Management System:**
- Hotel Directory Page — new page enabling users to add and view hotel inventory data, helping teams manage property details efficiently.

**Communication with Guests:**
- Unified Inbox — City and Sales agents can now reassign conversations back to their group (and not just back to the communication center as before).

#### <u>Enhancements & Bug Fixes</u>

- Enhancement *(Website)* — Tranzila payment form transformed to hosted fields for a consistent look on the new website.
- Fixed *(Airbnb)* — resolved blocked calendar issues for specific building properties on Airbnb.
- Fixed *(Digital Keys)* — fixed partial keypad image visibility for the guest view.
- Fixed *(Journeys)* — fixed sporadic stops in journey emails sent to guests.
- Fixed *(Inventory Management)* — ensured automatic copying of unit-level information when creating associated properties.
- Fixed *(Integrations)* — resolved errors in Intercom integration when bookings are created or updated.
- Fixed *(App)* — addressed issue of duplicate messages and incorrect order in the app chat.
- Fixed *(App)* — added missing push notification icon for Android devices.
- Fixed *(App)* — adjusted text size and capitalization for navigation bar text.
- Fixed *(App)* — resolved navigation issue from destination pages to neighborhood pages.
- Fixed *(Airbnb)* — fixed issue with Airbnb guests not receiving OCI verification codes when booking certain apartments.
