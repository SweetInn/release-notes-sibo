# Version P1.13

**Released:** Dec 05, 2024

### What's New?

#### <u>New Features</u>

- **Quotes Proposal for Guests** — enhancing and streamlining the conversion of leads to a direct booking.
- **Cleaner Lock Codes** — generate daily lock codes; start cleaning tasks and finish them automatically when the code is used.

#### <u>Website & App</u>

- New Bedrooms and Bathrooms Filter — enhanced filtering options for better property searches.
- Quote Proposal Page — guests can now view detailed proposals sent to them via a dedicated page.

#### <u>SIBO</u>

**Unified Inbox:**
- Create Quotes via Unified Inbox — directly create guest quote proposals during sales conversations.
- Automatic Quote Association — quotes are automatically linked to conversations for better tracking.
- Booking Association — when a quote converts to a booking, it will automatically be attached to the conversation.
- Assigned Conversations Filter — team members can filter assigned conversations for specific agents.
- Improved Search Functionality:
  - Search is now available for all users, including non-communication-center agents.
  - Extended to search for WhatsApp conversations using phone numbers or partial matches.
- Sticky Notes on Conversations — agents can create visible notes that appear prominently under the conversation subject for everyone.
- Sound Notification for Assigned Conversations — users receive sound alerts for newly assigned conversations.
  *(User must enable this option in their SIBO profile.)*

**Operations Platform — Cleaner Lock Codes:**
- Set/Replace Cleaning Lock Pin Codes — assign or replace lock pin-codes for cleaning companies.
- Daily Cleaner Codes — generate daily lock codes based on cleaning task schedules and the company's pin-code.
- Task Management via Lock Codes:
  - Start cleaning tasks when the lock pin-code is used.
  - Automatically finish cleaning tasks after the estimated cleaning time (+15 minutes) or when the lock is re-locked.
- Next Booking ETA & Guest Count — displays the estimated arrival time and number of guests for the next check-in when it occurs on the same day as the cleaning. *(Mobile Cleaning Task)*

**Automated Reports & Tasks:**
- In-Stay Review Submission Reports — automatically generate reports when guests submit in-stay reviews *(configurable by city)*.
- Check-In Tasks — automatically create a check-in task when the check-in method is set to in-person.
- Lock Dashboard Alerts — automatically generate reports from lock-related alerts on the dashboard.

#### <u>Enhancements & Bug Fixes</u>

- Fixed *(Services)* — late check-out was incorrectly marked as available on the check-out date.
- Fixed *(Chatbot)* — an issue was causing "undefined" text in chatbot check-out instructions for some conversations.
- Enhancement *(Hotel Directory)* — transportation line numbers in hotel directories are now editable.
- Fixed *(Airbnb)* — Airbnb messaging was remaining active when guests made a second direct reservation.
