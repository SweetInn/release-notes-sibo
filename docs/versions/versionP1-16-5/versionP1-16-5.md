# Version P1.16(5)

**Released:** Mar 09, 2025

![Version P1.16(5) - Updates]({{ "/assets/images/versionP1.16_5/Export4.jpg" | relative_url }})

### What's New?

#### <u>New Features</u>

**Marketing Promotions & Management:**
- Comprehensive Promotions Support — a structured interface for creating and managing promotions in SIBO, including four promotion types: generic, length of stay, last-minute, and early bird.
  - Retrieves banner details for accurate display.
  - Integrates promotions throughout the guest journey on the website and app, ensuring correct price calculations and booking details visibility.
  - SIBO quotes now support promotion-based pricing adjustments.

**Scores Dashboard:**
- External OTA Links Management — users can now manage OTA-related external links, including Booking.com URLs used for retrieving scores and data tracking.
- Scores Dashboard & Indications — a new dashboard calculates and aggregates scores from multiple sources, including NPS, Booking.com, and Airbnb, providing detailed insights. Includes threshold management with bulk updates, filters, and score indicators across SIBO pages including the multi-calendar, Unified Inbox sidebar, booking details, and quotes page.

**OTA Support:**
- Cancellation Policies for Hotels — enabled the option to set a separate cancellation policy for hotels.
- RU Messaging API Integration — Rentals United messaging integration now supports full chat mode through the Unified Inbox with channels including Booking.com, Expedia, and more. Includes automated booking association to ensure seamless communication with guests.

**Hotels Support:**
- Hotel Daily Cleanings — cleaning tasks are auto-generated daily based on hotel settings, ensuring efficient scheduling. A new **Do Not Disturb (DND)** status allows tasks to be flagged accordingly, improving operational control.
- Building Name Filter — users can now filter bookings by building or hotel name, making it easier to manage reservations across multiple properties.

#### <u>Additional Updates & Bug Fixes</u>

- Website — guests extending their stay were given an incorrect, overly low suggested price. This has been resolved.
- Website — added "No Elevator" amenity to the building page to clearly indicate properties without an elevator.
- SIBO — Clone User in User Management: enabled a new functionality allowing duplication of existing user profiles, assisting in serving new user requests with more accurate permission sets.
- Website — some units were shown on the building page despite having no availability. This has been resolved.
- Website — the inquiry form allowed users to select a check-in date after the check-out date. This has been fixed.
- Website — users were unable to slide through the image gallery on some devices. This has been fixed.
- SIBO — resolved an error occurring when disconnecting a property from Airbnb.
- SIBO — bank transfer amounts were incorrectly increased when modifying a Booking.com bank transfer reservation. This has been resolved.
