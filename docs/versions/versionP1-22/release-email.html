Reports and Tasks

  • Chained Tasks and Reports System manage an issue lifecycle completely with chained tasks, richer resolutions, re-opening when not fixed, and inventory usage attached to tasks. This gives full visibility and control from first report to final closure with accountable outcomes.
  • Nuki Smart Locks Automation low battery alerts on locks and keypads automatically create reports and maintenance tasks with provider level controls. This ensures timely replacements and fewer lockouts without manual triage.
  • Enhanced Task Display task cards show booking info, guest preparation details, and tags on mobile and desktop. This improves team context so work starts faster and errors are reduced.

Common Areas Management

  • Shared Spaces in Operations add and track building common areas in tasks, cleanings, and reports with filters for building and area. This organizes work across stairwells, rooftops, and lobbies so scheduling and reporting stay accurate.

Smarter Booking Filters

  • Presets and Custom Filters save frequently used booking filter combinations and manage your own custom filters. This shortens search time and keeps teams in flow.
  • Accommodation Type Filter use a unified accommodation type filter across key screens. This improves property categorization and speeds up discovery.

Website & App

  • New Connect Flow and Authentication introduce passwordless login and registration via secure email link. This simplifies access and reduces friction for new and returning users.
  • Enhanced Booking Flow Guest details are handled more cleanly for both registered and new users to improve conversion and reduce abandonment.
  • Partners Website & Quotes Preparation of the new website template and quote system for our SaaS partners.

Enhancements & Bug Fixes

  • Removal of Postpone the Postpone action was removed and replaced with clear rescheduling workflows and confirmations for in stay bookings. This eliminates confusion so task state and timing are always correct.
  • Password Field Eye Icon corrected the visibility toggle in the new connect flow. This prevents input mistakes and reduces retries.
  • AI Chatbot Accuracy improved response handling and relevance. This delivers more helpful answers and fewer escalations.
  • Credit Card Payment Failures fixed an issue where some scheduled credit card payments were repeatedly failing.
  • Language Reset on Redirect resolved a problem where clicking a building's unit link reset the selected language to English.
  • Missing OCI Translations added missing translations in the online check-in flow for a smoother guest experience.
