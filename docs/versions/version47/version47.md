# Version 47

**Released:** Jun 09, 2024

### What's New?

#### <u>Digital Keys</u>

<video width="100%" controls>
  <source src="{{ "/assets/Videos/version47_01.mp4" | relative_url }}" type="video/mp4">
</video>

- New digital keys experience (link and app).
- Show Digital Keys guide when user uses it for the first time.
- Added an additional last step to the Digital Keys guide.
- All guide screens are now fully functional while Digital Keys are active.
- Show Digital Keys locks after the guide ends.
- Enlarged the floor number shown on the locks.
- Replaced the expand/collapse circles with a direct full-page view of the lock.
- New **Block DKs** toggle in SIBO, allowing hosts to block Digital Keys access even after Online Check-In is completed.

#### <u>Booking</u>

- New **Additional IDs** section in SIBO — displayed when more IDs are collected than expected, allowing hosts to fix reservations stuck in Pending Online Check-In status.
- New content management page in SIBO for the website neighborhoods page.
- Keyboard improvements to booking search fields:
  - Auto-focus into the search field so the user can start typing immediately.
  - Enter to apply the search.
  - Escape to dismiss the filter.
  - Tab to move to the next field.
